# NeuralODE-Identification
## 项目简介
本项目基于PyTorch开发，主要实现了基于神经常微分方程的飞行器动力学模型辨识。

## 文件结构
- `preprocess.py`：数据预处理模块
- `data_processing.py`：数据处理模块
- `flight_scaler.py`：数据归一化模块
- `check_physics.py`：物理一致性校验模块
- `NeuralODEFunc.py`：神经网络、物理模型及ODE模块
- `pretrain.py`：预训练模块
- `train.py`：训练及参数设置模块
- `relobralo.py`: 动态多目标损失平衡模块
- `test.py`：测试模块
- `debug.py`：调试模块(功能不固定，主要用于检测问题)
- `generate_data.py`：测试数据生成(已弃用)

## 依赖
- `Python 3.9.22`
- `PyTorch 2.7.1`

## 运行
**首先要确保数据集的物理一致性。**
1. 运行`preprocess.py`，将原始数据文件转换为与变量协议一致的标准格式。
2. 运行`check_physics.py`，确保训练数据通过物理一致性校验。
3. 运行`data_processing.py`，执行数据处理流水线，生成数据集字典。
4. 运行`pretrain.py`，进行纯气动预训练，初始化模型参数。
5. 运行`train.py`，进行训练，保存模型参数并绘制训练曲线。
6. 运行`test.py`，进行测试，输出测试结果。

  - 滑动窗口框架对低频漂移具有天然鲁棒性
- 角加速度微分环节改用`savgol_filter`，提升对噪声数据的处理效果

### 2026-5-20 更新
- 优化噪声注入策略：仅对IMU数据（加速度/陀螺仪）添加10-20Hz带通噪声，移除姿态角和地速的噪声注入（其噪声频段与横向动力学频段重叠，会破坏训练标签的物理自洽性）
